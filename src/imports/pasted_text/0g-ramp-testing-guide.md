Overall style:

- Soft rounded cards
- Friendly and polished

- Light 2D/3D visual aesthetic
- Plenty of white space
- Easy to scroll and easy to read
- Use a soft tech/startup aesthetic

Canvas / layout:
- Create one long vertical desktop frame
- Width around 1440px
- Long scrollable height
- Use a centered content area around 960–1100px wide
- Add clear spacing between sections
- Use auto layout structure where possible

Typography:

- Big bold section headings
- Readable body text
- Keep paragraphs short and easy to scan
